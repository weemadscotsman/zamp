# MLM HERO - Game Design Document v1.0 🎮

## CORE GAMEPLAY LOOP

```
SCAM EVENT → MAKE CHOICE → STAT CHANGE → BRAINROT or REWARD → PROGRESS STORY
     ↑                                                            ↓
     └────────────────── NEW SCAM / MEGACORP ────────────────────┘
```

---

## PLAYER STATS

### Primary Stats

| Stat | Effect | How to Raise |
|------|--------|--------------|
| **Logic** | Detect fake claims, counter arguments | Education, Repentance Box, Pattern Recognition |
| **Confidence** | Resist pressure, make bold moves | Small wins, helping others, therapy |
| **Skepticism** | Unlock logic attacks, see red flags | Getting burned, research, Uncle Kev's tips |
| **Social Credit** | Repair relationships, warn others | Apologizing, helping victims, being honest |
| **Financial Health** | Escape debt, build real savings | Jobs, NOT reinvesting, budgeting |

### Secondary Stats

| Stat | Effect |
|------|--------|
| **Stress** | Too high = bad decisions forced |
| **Trauma** | Tracks scars, unlocks survivor dialogue |
| **Innocence** | Starts high, drops with experience |
| **Cynicism** | Too high = can't trust anyone (bad) |

### The Balance
- **Naive (0 Skepticism)** → Easily scammed, manipulated
- **Healthy Skeptic (50 Skepticism)** → Can spot scams, still hope
- **Cynical (100 Skepticism)** → Can't accept help, paranoid

## CONVERSATION BATTLE SYSTEM

### Combat Flow
```
ENEMY ATTACK → PLAYER CHOICE → OUTCOME → NEXT TURN
```

### Enemy Attack Types

| Attack | Damage | Effect |
|--------|--------|--------|
| **Toxic Positivity** | -10 Confidence | "Good vibes only!" |
| **Fake Success Story** | -15 Logic | Shows rented Lambo pics |
| **Gaslighting** | -20 Confidence | "You're just negative" |
| **Love Bombing** | +Stress, -Skepticism | Fake friendship |
| **Urgency Timer** | Forces quick decision | "Only 3 spots left!" |
| **Income Claim** | -25 Financial Health | "I made $50k this month!" |
| **Family Guilt** | -30 Social Credit | "Don't you want to provide?" |

### Player Response Options

#### Basic Responses (Always Available)
1. **Agree** - Take damage, gain temporary benefits
2. **Deflect** - Avoid damage, no progress
3. **Question** - Mild counter, requires Logic
4. **Reject** - Strong counter, requires Confidence

#### Unlockable Responses (Require Stats)
| Response | Required Stat | Effect |
|----------|---------------|--------|
| **Show Math** | 30 Logic | Reveals truth, stuns enemy |
| **Use Sarcasm** | 40 Confidence | Deflects + deals ego damage |
| **Deploy Evidence** | 50 Skepticism | Massive damage, ends battle |
| **Irish Goodbye** | 30 Confidence | Escape instantly |
| **Grey Rock** | 40 Trauma | Immune to emotional attacks |
| **Call Mom** | 50 Social Credit | +Confidence buff |

### Special Battle Modes

#### THE ZOOM CALL (Boss Fight)
- Multi-phase battle
- Must survive upline presentation
- Interruption mechanics
- Group chat chaos as backup

#### THE CONFERENCE (Final Boss)
- Hundreds of cultists present
- Must convert audience while fighting boss
- Public speaking meter
- Risk of being shouted down

## BRAINROT BANISHMENT SYSTEM

### Trigger Conditions
Player gets banished for:
- Blindly reinvesting profits
- Ignoring warning signs
- Recruiting friends/family
- Clicking urgency timers
- Using "trust the process" dialogue

### The Realm
Vertical scrolling hell of:
- Fake luxury TikToks
- Crypto bro rants
- MLM boss babes in bathrooms
- AI testimonial generators
- "Passive income" compilations

### Mini-Game: Spot The Red Flag

**Gameplay:**
- Video autoplays (5-10 seconds)
- Player must tap red flags before video ends
- 3 strikes = Brainrot meter fills

**Red Flags To Spot:**
- 📊 Fake profit screenshots
- ⏰ "Limited time!" countdown
- 🏠 Stock footage mansion
- 🚗 Rented luxury car
- 💰 Guaranteed returns
- 👥 "Join my tribe"
- 📱 "Drop a 💎 if you're ready"
- 🧠 "Mindset is everything"

### Brainrot Meter Effects

| Level | Effect |
|-------|--------|
| 0-25% | Normal |
| 25-50% | -10 Logic, screen slightly glitches |
| 50-75% | -25 Logic, dialogue options get dumber |
| 75-100% | **BRAINROTTED** - Must complete Repentance Box |

### Brainrot Status Effect
```
🧠 BRAINROTTED
- Logic reduced by 50%
- Auto-select worst dialogue options
- Screen constantly glitches
- Achievement: "Touched Too Much Grass"
```

## REPENTANCE BOX SYSTEM

### Entry Trigger
- After Brainrot Banishment
- After major scam collapse
- Player can voluntarily enter for buffs

### The Experience
```
REMOVING INFLUENCER RESIDUE...
PURGING MOTIVATIONAL QUOTE MALWARE...
RESTORING COMMON SENSE.dll...
INSTALLING SKEPTICISM PATCH...
```

### Content Types

#### 🐕 Animal Chaos Compilation
- Puppies discovering stairs
- Cats losing fights with curtains
- Ducks waddling with criminal intent
- Goats screaming at nothing

#### 🌧️ Calm Ambience Mode
- Rain on window
- Fireplace crackling
- Forest sounds
- Gentle lo-fi beats

#### 💤 Soft Reset Mode
- Mandatory 30-second breathing
- No decisions allowed
- Just... existing

### Recovery Effects

| Duration | Effect |
|----------|--------|
| 30 sec | Remove Brainrot, +10 Logic |
| 1 min | +20 Logic, +Confidence buff |
| 2 min | +Skepticism, unlock special dialogue |
| 5 min | **ZEN MASTER** - Immune to urgency timers for 1 hour |

### Secret Feature
If player tries to skip too fast:
```
WARNING: Residual Hustle Energy Detected
Recommended Additional Puppies
[Watch 3 more dog videos to proceed]
```

## MEGACORP INFILTRATION SYSTEM

### The Cover
Player works as "Opportunity Development Associate"

### Daily Cycle
```
MORNING: Corporate meeting (comedy/horror)
MIDDAY: "Work" tasks (sabotage opportunities)
AFTERNOON: Risk assessment (suspicion check)
EVENING: Secret actions (help victims)
```

### Suspicion Meter
| Level | Effect |
|-------|--------|
| 0-20% | Safe, bonuses available |
| 20-40% | Watch what you say |
| 40-60% | Random "loyalty tests" |
| 60-80% | Under surveillance |
| 80-100% | **EXPOSED** - Fired or captured |

### Sabotage Actions

| Action | Suspicion Risk | Victims Saved |
|--------|----------------|---------------|
| Edit script to add disclaimer | Low | 10-20 |
| Leak victim list to authorities | Medium | 50-100 |
| Delay "maintenance" | Low | 20-30 |
| Plant evidence for journalists | High | 200+ |
| Warn individual victim | Very Low | 1 (but personal) |
| Hack dashboard show real numbers | Very High | 500+ |

### Moral Dilemmas

**The New Recruit**
- New hire is about to sign up
- They look EXACTLY like player did
- Choice: Blow cover to save them, or stay hidden?

**The Promotion**
- Get offered management position
- Double salary
- "You'll have more influence to change things..."
- Is it compromise or infiltration?

**The Child Database**
- Discover recruitment targets include minors
- Must act NOW regardless of risk
- No perfect solution

## ECONOMY SYSTEM

### Money Sources (Legit)
| Source | Income | Time | Stress |
|--------|--------|------|--------|
| Gig delivery | $50/day | High | Medium |
| Freelance work | $100/day | Medium | Low |
| Part-time retail | $80/day | High | High |
| Tutoring | $60/day | Low | Low |
| Selling MLM stock | Variable | Low | High |

### Money Drains
| Expense | Cost |
|---------|------|
| Rent | $800/month |
| Food | $300/month |
| Phone | $50/month |
| "Investment opportunity" | $500+ (scam) |
| Therapy | $100/session (good) |

### Debt System
- Start with $2,000 MLM debt
- Interest compounds if ignored
- Debt collectors call (stress events)
- Bankruptcy option (resets but scars)

## RELATIONSHIP SYSTEM

### Key NPCs

| NPC | Initial State | Can Repair? | Benefit |
|-----|---------------|-------------|---------|
| Mom | Disappointed | Yes | Moral support |
| Best Friend | Blocked you | Yes | Reality check |
| Ex-Partner | Angry | Maybe | Closure |
| Co-worker | Suspicious | Yes | Job help |

### Repair Mechanics
- Genuine apology (not excuse)
- Show changed behavior
- Help THEM with something
- Time + consistency

### Warning System
As player gains Skepticism, can warn others:
- Prevents them from joining scams
- Builds Social Credit
- Sometimes they don't listen (realistic)

## PROGRESSION SYSTEM

### Skill Tree: Survivor Path
```
LEVEL 1: Basic Skeptic
  └─ Pattern Recognition
  └─ Red Flag Detection

LEVEL 2: Informed Citizen  
  └─ Math Literacy
  └─ Evidence Evaluation
  └─ Emotional Boundary

LEVEL 3: Community Defender
  └─ Public Speaking
  └─ Victim Support
  └─ System Analysis

LEVEL 4: Scam Immune
  └─ Bullshit Radar
  └─ Hope Without Naivety
  └─ Pay It Forward
```

### Achievement Ideas

**Story Achievements:**
- 🔥 "Touched The Stove" - First scam burn
- 🧠 "Installed Common Sense" - Complete first Brainrot
- 🐕 "Palette Cleansed" - Use Repentance Box 10 times
- 🕵️ "Pattern Spotter" - Recognize 5 scam similarities
- 🏢 "Inside Job" - Infiltrate megacorp
- 📢 "The Receipts" - Expose the system
- 💚 "Welcome To The Club" - Complete game

**Secret Achievements:**
- 👑 "You Became The Boss" - Join the dark side

- 🏈 "Bake Sale Queen" - Crystal's redemption arc
- 📊 "One Shelf" - Help Gary finish project
- 🧅 "Emotionally Seasoned" - Max Trauma but healthy

**Meme Achievements:**
- 💎 "Diamond Hands (For Leaving)" - Exit before collapse
- 🐕 "Recruited The Dog" - Sorry cousin
- 🧠 "Brainrot Immunity" - Never get banished
- 📉 "0.1%" - Join the actual profitable tier (impossible?)

## UI/UX DESIGN

### Main Screen
- Top: Financial Health bar
- Left: Stats (Logic, Confidence, Skepticism)
- Right: Social Credit, Stress
- Center: Scene/dialogue
- Bottom: Choice buttons

### Brainrot Realm
- Full screen vertical video format
- Rapid glitch effects
- Overwhelming visual noise
- Anxiety-inducing by design (but funny)

### Repentance Box
- Soft colors, gentle animations
- Slow, calm movements
- Breathing indicator
- "Take your time" messaging

### Megacorp Office
- Corporate blue/grey
- Motivational posters everywhere
- Fake smiles on NPCs
- Pyramid shapes hidden in architecture

## ART STYLE DIRECTION

### Option A: Corporate Horror Cartoon
- Over-polished smiling faces
- Fake luxury aesthetic
- Glitch effects when truth revealed
- Papers Please meets Disco Elysium

### Option B: Influencer Glitch Hell
- Neon overload
- Overstimulating transitions
- Ring light halos around scammers
- Vertical video format for battles

### Recommended: HYBRID
- Normal world: Corporate Horror
- Brainrot Realm: Influencer Glitch
- Repentance Box: Soft watercolor
- Megacorp: Both combined

## AUDIO DIRECTION

### Sound Design
- **Normal scenes:** Ambient city, notifications
- **Battles:** Rising tension, heartbeat
- **Brainrot:** Cacophony of voices, glitch static
- **Repentance:** Soft piano, nature sounds
- **Megacorp:** Elevator music with wrong notes

### Music Themes
- **Menu:** Lo-fi chill (ironic)
- **Act 1:** Upbeat false hope
- **Act 2:** Somber realization
- **Act 3:** Tense espionage
- **Ending:** Warm, grounded

## TECH STACK RECOMMENDATION

- Built-in UI tools

- Faster prototyping
- Easier for non-gamedev
